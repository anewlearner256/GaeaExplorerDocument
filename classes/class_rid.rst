RID
===

:ref:`Resource<class_Resource>` 的唯一 ID 的句柄。

描述
----

RID类型用于访问资源的唯一整数ID。它们不透明，这意味着其本身并不授予对相关资源的访问。它们被低级别的服务类使用，如\ :ref:`VisualServer<class_VisualServer>`\ 。
