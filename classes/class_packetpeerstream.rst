PacketPeerStream
================

**Inherits:** :ref:`PacketPeer<class_PacketPeer>` **<** :ref:`Reference<class_Reference>` **<** :ref:`Object<class_Object>`

在StreamPeer上使用PacketPeer的包装器。

描述
----

PacketStreamPeer提供了一个包装器，用于在流上使用数据包工作。这允许在StreamPeer上使用基于数据包的代码。PacketPeerStream在StreamPeer上实现了一个自定义的协议，所以用户不应该直接读或写到被包装的StreamPeer上。
