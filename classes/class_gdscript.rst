一个用 GDScript 编程语言实现的脚本。

描述
----

用GDScript编程语言实现的脚本。该脚本扩展了实例化该对象后其所有对象的功能。

\ :ref:`new<class_GDScript_method_new>` 创建脚本的新实例。如果一个对象的类与脚本的基类之一匹配，则 :ref:`Object.set_script<class_Object_method_set_script>` 会扩展该对象。
