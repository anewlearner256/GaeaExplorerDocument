VisualInstance

所有可视3D节点的父节点。

描述
----

``VisualInstance``\ 用于连接一个资源和一个视觉显示。所有的可视化三维节点都继承自\ ``VisualInstance``\ 。你不应该直接访问\ ``VisualInstance``\ 的属性，一般来说，它们是由继承自\ ``VisualInstance``\ 的节点访问和管理的。\ ``VisualInstance``\ 是\ :ref:`VisualServer<class_VisualServer>`\ 实例的节点表示。
