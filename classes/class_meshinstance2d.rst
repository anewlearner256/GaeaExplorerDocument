描述
----

用于在2D中显示\ :ref:`Mesh<class_Mesh>`\ 的节点。可以通过编辑器工具栏上的工具从现有的\ :ref:`Sprite<class_Sprite>`\ 构建。选择 "Sprite"，然后选择 "转换为Mesh2D"，在弹出的窗口中选择设置并按下 "创建Mesh2D"。
