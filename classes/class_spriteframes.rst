SpriteFrames
============

**Inherits:** :ref:`Resource<class_Resource>` **<** :ref:`Reference<class_Reference>` **<** :ref:`Object<class_Object>`

AnimatedSprite 的精灵帧库。

描述
----

:ref:`AnimatedSprite<class_AnimatedSprite>` 的精灵帧库。包含可用于播放的帧和动画数据。

\ **注意：**\ 你可以通过创建带有 ``_normal`` 后缀的 ``SpriteFrames`` 来关联一组法线贴图。例如，同时存在两个分别叫做 ``run`` 和 ``run_normal`` 的 ``SpriteFrames`` 资源，那么 ``run`` 中的动画就会使用法线贴图。
