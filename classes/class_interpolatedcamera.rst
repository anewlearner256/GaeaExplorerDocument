描述
----

*已经弃用（将在GaaeExplorer 4.0 中删除）。*\ 插值相机是一种\ :ref:`Camera<class_Camera>`\ ，可以平稳地移动，以匹配目标节点的位置和旋转。

如果它不是\ :ref:`enabled<class_InterpolatedCamera_property_enabled>`\ 或没有有效的目标集，InterpolatedCamera 的行为就像一个正常的相机。
