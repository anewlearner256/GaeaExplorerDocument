ReferenceRect
=============

**Inherits:** :ref:`Control<class_Control>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

GUI的引用框架。

描述
----

矩形框，仅在其矩形周围显示 :ref:`border_color<class_ReferenceRect_property_border_color>` 边框颜色。 ``ReferenceRect`` 没有填充 :ref:`Color<class_Color>`\ 。如果你需要显示填充纯色的矩形，请考虑使用 :ref:`ColorRect<class_ColorRect>` 。
