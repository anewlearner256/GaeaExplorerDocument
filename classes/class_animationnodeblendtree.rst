:ref:`AnimationTree<class_AnimationTree>`\ 节点资源，其中包含许多混合类型节点。

描述
----

该节点可以包含任何其他混合类型节点的子树，例如mix，blend2，blend3，一个shot等。这是最常用的根之一。
